# CYBER GYM — راهنما

## فایل‌ها (هر ۴ تا رو با هم آپلود کن)

- `index.html` — خود برنامه (عکس سازنده هم دیگه داخل همینه، نیازی به فایل جدا نداره)
- `manifest.json` — مشخصات نصب روی صفحه اصلی
- `service-worker.js` — کارکرد آفلاین
- `icon.svg` — آیکون برنامه

(فایل `creator.jpg` رو هم برات گذاشتم ولی برنامه دیگه بهش نیاز نداره — عکس مستقیم داخل index.html جاساز شده تا همه‌جا درست نمایش داده بشه.)

## آپلود رایگان روی گیت‌هاب (GitHub Pages)

۱. وارد github.com شو و یه اکانت رایگان بساز (اگه نداری)
۲. بالا سمت راست روی + بزن → **New repository**
نام بده مثلاً `cyber-gym` → گزینه **Public** → **Create repository**
۳. توی صفحه ریپو، روی **Add file → Upload files** بزن
۴. ۴ فایل بالا رو بکش و بنداز داخل کادر → پایین صفحه **Commit changes**
۵. برو به تب **Settings** بالای ریپو → از منوی سمت چپ **Pages**
۶. زیر «Build and deployment» → Source: **Deploy from a branch**
Branch: **main** و فولدر **/ (root)** → **Save**
۷. حدود ۱-۲ دقیقه صبر کن، یه لینک مثل این بهت میده:
`https://USERNAME.github.io/cyber-gym/`
۸. همین لینک رو برای رفیقات بفرست و روی گوشی خودت باز کن

### نصب روی صفحه اصلی گوشی

- آیفون: لینک رو با سافاری باز کن → دکمه Share → **Add to Home Screen**
- اندروید: لینک رو با کروم باز کن → منوی سه‌نقطه → **Add to Home screen** یا **Install app**

## چیزهای جدید این نسخه

- یه ترنزیشن ساده هنگام ورود به برنامه (اسم CYBER GYM با یه لودینگ کوتاه قبل از باز شدن صفحه اصلی)
- وقتی همه‌ی تمرین‌های شنبه تا پنجشنبه تیک بخوره، چند ثانیه بعد خودکار همه‌چی ریست می‌شه
  تا برای هفته جدید آماده باشه (یه پیام تبریک هم نشون داده می‌شه)
- بخش جدید «بعد از باشگاه»: شمارنده تخم‌مرغ و سیب‌زمینی آب‌پز با + و −، به همراه
  حساب تقریبی پروتئین دریافتی (اعداد تخمینی‌ان، نه یه منبع علمی دقیق)
- بخش پیشرفت: وقتی حداقل ۲ تا عکس ثبت کرده باشی، یه «مقایسه دو عکس» با اسلایدر قبل/بعد
  اضافه می‌شه که می‌تونی هر دو تا عکس دلخواه رو انتخاب و کنار هم (با خط کشیدنی) ببینی،
  به همراه تعداد روزهای فاصله بینشون
- لیست ورزشکاران از اول با ۴ نفر پر شده (خودت + ۳ نفر دیگه) — هر کدوم رو می‌تونی
  ویرایش/حذف کنی یا نفر جدید اضافه کنی (عکس از روی اسم خودت برداشته شده، فقط توی
  «درباره ما» می‌مونه)
- بخش «درباره ما»: شماره الان به اسم برنامه‌نویس ثبت شده (نه مربی)، و عکس سازنده
  مستقیم داخل فایل جاساز شده تا همه‌جا درست لود بشه

### درباره مقایسه عکس‌ها

نمی‌تونم درصد دقیق تغییر بدن رو خودکار حساب کنم و بگم کجا بیشتر تغییر کرده — این نیاز به
یه تحلیل هوش مصنوعی واقعی (مدل بینایی کامپیوتر) داره که فقط وقتی کار می‌کنه که برنامه از
طریق خود Claude.ai اجرا بشه، نه وقتی به‌صورت فایل مستقل دانلود و روی گیت‌هاب میزبانی بشه
(چون اون اتصال امن به مدل هوش مصنوعی فقط داخل محیط Claude در دسترسه، نه روی یه سایت
استاتیک عمومی). به‌جاش یه اسلایدر مقایسه‌ی قبل/بعد ساختم که خودت می‌تونی با چشم خودت
راحت‌تر تغییرات رو ببینی — دقیق‌تر و صادقانه‌تر از یه عدد ساختگیه.

## نکته درباره داده‌ها و اشتراک‌گذاری

این یه برنامه‌ی کاملاً سمت کاربره (بدون سرور)، یعنی:

- لینک رو با هرکی بخوای به اشتراک می‌ذاری و خودش روی گوشی خودش باز و نصب می‌کنه
- ولی تیک تمرین‌ها، عکس‌های پیشرفت، شمارنده‌ی بعد از باشگاه و لیست ورزشکارها روی
  **همون گوشی** ذخیره می‌شن، نه روی یه سرور مشترک — یعنی تیک‌هایی که تو می‌زنی برای
  رفیقت دیده نمی‌شه و برعکس، هرکس برنامه‌ی خودش رو جدا پیگیری می‌کنه.
- اگه بخوای تیک‌ها واقعاً بین چند نفر مشترک و همگام (sync) باشه، این به یه دیتابیس/سرور
  جدا نیاز داره (مثلاً Firebase رایگان)؛ هنوز برات اضافه نکردم — بگو تا انجامش بدم.

برای پاک نشدن داده‌ها، کش/دیتای سایت رو از تنظیمات مرورگر پاک نکن.
